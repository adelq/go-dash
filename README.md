# go-dash (alpha)

A low-overhead monitoring web dashboard for a GNU/Linux machine. Simply 
drop-in the app and go!


## Documentation

### Distribution-specific information

`/issue`
```json
{
  "distro": "Ubuntu 14.04.4 LTS",
  "kernel": "3.13.0-85-generic"
}
```
